PD2AF converter is an open-source application designed to enable communication between the Process Description and the Activity Flow languages of the Systems Biology Graphical Notation (SBGN) standard. The purpose of the tool is to translate PD to AF in such a way so it produces the result comparable to manually designed AF diagrams and the corresponding logical models.
