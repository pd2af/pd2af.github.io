Welcome to the PD2AF project website!

The PD2AF converter is an open-source application designed to translate the Process Description (PD) format to the Activity Flow (AF) format of the Systems Biology Graphical Notation standard.

The project includes two main components:

  • The description of the conversion rules and their visualisation to facilitate systematic development and discussions on alternative translation scenarios.
  • The online converter that transforms Process Description SBGN-ML to Activity Flow SBGN-ML.

We also collect manually-verified pairs of PD-AF maps of different complexity to provide examples and prepare the basis for improving the converter functionalities.
